# LetLib
Utility library. Created for LetCraft.pl

## Maven

```xml
<repository>
  <id>letcraft-repo</id>
  <url>https://repo.letcraft.pl/releases/</url>
</repository>

<!-- Bukkit -->
<dependency>
  <groupId>pl.letcraft</groupId>
  <artifactId>letlib-bukkit</artifactId>
  <version>1.0.2</version>
</dependency>


<!-- Bungee -->
<dependency>
  <groupId>pl.letcraft</groupId>
  <artifactId>letlib-bungee</artifactId>
  <version>1.0.2</version>
</dependency>
